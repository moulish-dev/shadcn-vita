# Install

### Python version
- **Python 3.11** recommended (project README target).

### Steps

    :::python
    git clone https://github.com/moulish-dev/vita.git
    cd vita
    python3.11 -m venv .venv
    source .venv/bin/activate   # Windows: .venv\Scripts\activate
    pip install -r requirements.txt
    pip install -e .
