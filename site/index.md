### What is VITA?
VITA is a toolkit that unifies multiple **Text-to-Speech** backends behind a simple API and CLI.

- **Use cases:** assistants, accessibility, robotics (Raspberry Pi), creators, batch audio tools  
- **Highlights:** clean API, `.wav` outputs, quick install, local-first

> New here? Start with the **Quick Start** to generate your first audio in 2 minutes.
